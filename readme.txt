Get information in real time to find breaking stories with openFeed


== Description ==
openFeed is a real time feeds service showing you currently stories being published around the world. 


Features:

* Real time tracking service, find stories when they break
* See current trends and hot stories making the rounds with our Trends feed


Available in these languages:

* English 



If you discover a problem with this plugin, please report it at https://wordpress.org/tags/openfeed


For further information and or API development, visit
http://www.openfeed.io

== Installation ==

1. Download
2. Upload to your `/wp-contents/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
